当产品出现以下质量问题，我们将免费为您办理换货服务

  • 产品在制作过程沾有严重的污垢、油墨等脏物。
  • 产品变形，起壳、起泡、起皱。
  • 画面严重爆网、掉墨、严重套位不准。
  • 产品明显裁歪。
  • 印刷或发货错误，与您的网络订单要求不符。
  • 产品数量缺少或缺页。

发现质量问题，请于签收商品之日起24小时内与客服联系，我们确认后将在第一时间与您协商处理。

下述此类非产品质量问题，不能享受售后服务：

  • 因环境光、电脑显示器色差、纸张等原因而导致的电脑上的图片与实物之间有色差。
  • 上传的相片由于像素低、曝光不足而导致噪点增多、曝光过度等问题。
  • 由客户原因引起的产品污垢，损坏，破损等。
  • 客户因主观因素而不喜欢、觉得精度不高、觉得与想象差别较大等问题。
  • 自行更改原有设计模板导致出现问题。

常见问题FAQ
Q : 为什么我冲印出的照片这么模糊？
A : 拍摄过程中由于手抖、光线不足，相机、手机摄像头像素低、曝光不足而导致噪点增多、曝光过度都是造成照片模糊的原因。在QQ空间、微博、微信上下载的图片或经过美图秀秀制作的图片，像素都比较低，也是造成照片模糊的原因，所以这类图片不建议印刷。
Q : 为什么印刷出来的照片与我的相片颜色有差别？
A : 由于各牌子、型号电脑显示器屏幕特点不同，每台电脑显示器显示出的照片颜色都可能存在差别,而不同的纸张,也会造成偏色,我们不能承诺100%还原您电脑显示器屏幕上的色彩。但我们承诺会使用优质的印刷器材为您提供优质的印品。
Q : 在POCOPRINT上付款是安全的吗？
A : 绝对是安全的！我们使用的是第三方支付系统，例如支付宝，通过支付宝的系统能够最大限度保证支付过程的安全，而且我们已建立了一个相对完善的服务保障系统，所以您不需要担心您的资金安全，请您放心。
Q : 支付失败怎么办？
A : 在支付过程中，可能会出现金额不足、意外断线等状况，而导致支付不成功。如果出现支付失败，您只需进入 “我的订单”，找到相应订单并点击“付款”，便可重新支付。
Q : 如何多次下订单只付一次运费？
A : 只要第一次下订单后五小时内再次定制，不更改收货人，手机号码和收货地址，系统将自动减去运费，成功下单后还即送五元消费券。
